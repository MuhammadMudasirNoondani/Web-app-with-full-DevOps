# Web-app-with-full-DevOps
web app with full DevOps Intership project recgenz 
A production-ready  web application demonstrating end-to-end DevOps practices.
Built with Go (net/http), containerized using Docker multi-stage builds with a
distroless image, and deployed to Kubernetes (EKS Fargate) via Helm charts.
CI/CD is fully automated with GitHub Actions and GitOps delivery powered by Argo CD.
